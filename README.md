# 🌍 ESG Analytics Dashboard

A professional ESG (Environmental, Social, Governance) analytics dashboard comparing sustainability metrics of Infosys and Reliance using real-world BRSR data.

---

## 🚀 Live Demo
👉 https://esg-dashboard-gojvjsmxrxazddqr5mp5lh.streamlit.app/

---

## 📊 Features

- 📈 KPI comparison (Gender Diversity, Attrition, Governance)
- 🎛️ Interactive metric selection
- 📥 Download dataset functionality
- 🏆 Performance insights & decision indicators
- 📊 Clean data visualizations
- 🌐 Fully deployed web application

---

## 🧠 Key Insights

- Infosys shows significantly higher gender diversity  
- Reliance has lower attrition (more stable workforce)  
- Governance diversity remains low across both companies  
- Climate risk is common across industries
